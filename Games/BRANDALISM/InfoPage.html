GENERAL INFO

This app was made for a school project at SintLucas

Students from the art direction and software direction had to make teams and create a game or webpage

The art students we're in charge of what the project would be


Me and another software dev student got assigned to team brandalism

The project was to make a mobile / web based game used for advertizing

Advertisements would be in a graffiti or street-art style, and could be scanned with your phone to play a game


We picked tetris as the game to code for this demo version

The AR system works but the "poster" we used is no longer available

However the game is still playable in the offline / non-AR mode


If you need something to compare it to, the Red bull tetris ad campaign is very similar


TECHNICAL INFO

This game uses Unity

If you wish to play the game download the app to your phone from the link above

The game will slowly become faster as you gain more score

GAMEPLAY

Its tetris on mobile

Swipe the screen to move the blocks

Tap the screen to rotate the blocks

Once the app is started tap the eye icon to switch to non-AR mode

Try to get a high score
