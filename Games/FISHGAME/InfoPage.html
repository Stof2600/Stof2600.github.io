GENERAL INFO

This game was made for a school practice exam, the focus was to learn proper documentation so the game itself is very simple

The games design was done by the teachers


TECHNICAL INFO

My version uses Godot as its game engine and all the pixel art is done by me

I also used this project to test out the particle system godot has

GAMEPLAY

Clicking above the aquarium will spawn food for the fish to eat at the cost of 1 coin

Fish will drop coins once grown up

More fish can be bought in the store

The enemies that apear can be killed by clicking them a few times
