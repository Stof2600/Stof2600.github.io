ORIGIN

This was one of my first projects from when I started in game development

Virtual reality was very popular at the time and my school had a HTC Vive and PC for me to work on

The original plan was to make a game where you can craft items and weapons yourself

The player could do this by finding parts or materials in levels and would then have to process them in some kind of hub world


MESSING AROUND IN VR

None of the original idea is in the last build of the game

I made a pistol to try coding weapons and found it really fun to mess with

After that the project shifted into trying to make realistic Firearms


FIREARMS

Since my 3D modeling and animation skills aren't that good all of the weapons are driven with physics and script

Because its an early project the code isn't all that good, and all the weapon script was done in a single file

The grabbing and reloading is also very imprecise up to the point where some weapons take a few attempts to reload


ADITIONAL STUFF

I worked on this project for about a year

While weapons where the main priority I did work on some other stuff

There is an AI system for enemies in the game which I remember being very intricate and annoying to write

However it did work and while testing the AI seemed pretty smart
